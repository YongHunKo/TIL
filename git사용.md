# git
> 분산 버전 프로그램
> **백업**,**복구**,**협업**을 위해 사용
## CLI
> command line interface, 터미널을 통해 사용자와 컴퓨터가 상호작용하는 방식
### 터미널 명령어 정리
`mkdir` = 폴더 생성
`touch` = 파일 생성(*.txt , *.md 등등)
`ls`    = 현재 폴더의 파일 목록 출력
`cd`    = 다른 폴더로 이동
`rm`    = 파일 삭제 / 폴더 삭제 (-r로 사용하기도 함)
